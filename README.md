# AI_Andrej_Karpathy

- create environment:
    `sudo python3 -m venv ~/venvs/Andrej_Karpathy/01`

- activate virtual environment:
    `source ~/venvs/Andrej_Karpathy/01/bin/activate`

- deactivate environment:
    `deactivate`

- install requirements
    `python3 -m pip install -r ./requirements.txt`
